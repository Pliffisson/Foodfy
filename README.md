## 🗂 Como baixar o projeto:

```bash
  # Clonar o repositório
  $ git clone https://github.com/Pliffisson/foodfy.git


  # Entrar no diretório
  $ cd foodfy


  # Instalar as depedências do projeto
  $ yarn install


  # Iniciar o projeto
  $ yarn start
```
